# DLSS ENABLER MANAGER

A tool developed by **Nodix Tech** to help PC gamers easily manage and install Frame Generation and Streamline mods.

### Key Features:
- **Smart Scanner:** Automatically detects games across all drives (Steam, Epic, and manual folders).
- **Compatibility Check:** Verifies Streamline version (2.4+) and required DLLs.
- **Safety First:** Automatic backup of original files before any modification.
- **Easy Rollback:** One-click uninstall to restore original game files.

### How to use:
1. Run the application and scan for your games.
2. Check the compatibility status.
3. Click "Install Mod" and enjoy the FPS boost!
